# Simple Linear Regression – Marketing ROI Analysis

## Project Overview
This project applies data science techniques to evaluate a corporate marketing dataset and construct a statistically valid Simple Linear Regression model. The business objective is to analyze historical expenditures across three prominent marketing channels—**TV, Radio, and Social Media**—and determine which channel drives the highest, most predictable Return on Investment (ROI) relative to overall **Sales**. 

By translating raw performance data into explicit statistical variables, this analysis equips non-technical stakeholders with the necessary insights to optimize corporate marketing budget allocations.
